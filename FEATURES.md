# iFocus 功能列表

## 核心功能

### 🍅 番茄钟计时
- 标准番茄工作法：25分钟专注 + 5分钟休息
- 长休息：每4个番茄后15分钟长休息
- 自定义长休息时间段（如午餐、晚餐、睡觉）
- 声音提示和系统通知

### ⏰ 自动整点模式
- 每半小时自动循环（0-25分钟专注 + 25-30分钟休息）
- 自动检测并进入长休息时间段
- 契合自然工作节律
- 实时同步到当前时间段

### 📋 任务管理
- 创建任务并预估所需番茄数
- 追踪完成的番茄数
- 任务完成标记
- 清理已完成任务
- 本地持久化存储

### 📌 Mini 模式
- 小巧浮窗（200x35像素）
- 始终置顶显示
- 鼠标拖动移动位置
- 双击退出 Mini 模式
- 显示剩余番茄数和进度
- Windows 圆角窗口优化

### 🔐 GitHub 数据同步
- GitHub OAuth 登录
- 数据存储在私有 Gist
- 上传/下载数据
- 多设备数据共享
- Token 方式备用登录

### 🎨 界面设计
- 根据模式自动切换主题色：
  - 专注模式：橙红色
  - 短休息：青绿色
  - 长休息：紫蓝色
- 平滑的颜色过渡动画
- 现代化的 UI 组件
- 响应式布局

### 🖥️ 系统集成
- 系统托盘图标
- 最小化到托盘
- 关闭按钮最小化而非退出
- 托盘菜单快捷操作
- 单实例应用（防止重复启动）

## 高级功能

### 🌴 自定义长休息时间段
- 添加/删除时间段
- 自定义名称和图标
- 设置开始/结束时间
- 启用/禁用时间段
- 支持跨午夜时间段
- 按时间自动排序

### 📊 番茄数统计
- 今日剩余番茄数显示
- 根据长休息时段智能计算
- 颜色渐变指示（绿→黄→橙）
- Mini 模式实时显示

### 🔔 通知系统
- 计时器完成通知
- 模式切换提示
- 自动模式时间段提醒
- GitHub 登录状态通知
- 数据同步状态提示

### ⚙️ 设置管理
- 可视化设置界面
- 长休息时间段配置
- 图标分类选择器
- GitHub 同步设置
- 一键导入/导出数据
